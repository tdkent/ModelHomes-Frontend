# Model Homes of the Golden Gate International Exposition, 1939-40

This documentation is for the **React frontend** of the content-driven website chronicling the history of the model homes built in the San Francisco Bay Area for the 1939-40 World's Fair.

## Tech Stack

- Language: TypeScript
- UI Library: React w/Vite
- Routing: React Router
- Lint &amp; Format: Biome w/Husky pre-commit
